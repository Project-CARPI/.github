# Welcome to Project CARPI!
## What does CARPI stand for?
**C**ool **A**cademic **RPI**!
## What is this?
The goal of this project is to aid RPI students in their academic endeavors in any meaningful way, shape, or form.

We're currently working on a course planner web application, which you can check out under our repository list!

Additionally, check out the [CARPI Bot](https://github.com/CARPI-Bot/CARPI-Bot) project that we originally formed our group around!
